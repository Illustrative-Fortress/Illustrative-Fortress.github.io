Installation instructions:

  • You must have Source SDK Base Multiplayer 2013 installed to play Illustrative Fortress.
  • Illustrative Fortress has to be installed in the same drive as steam, then put it in the source mod folder, example: C:\Program Files (x86)\Steam\steamapps\sourcemods\if
  • Before running Illustrative Fortress, right click on Source SDK Base Multiplayer 2013 then click properties and go to the Game Versions & Betas and choose prerelease(recommended) or Default Public Version. Restart Steam after the SDK branch finishes changing.
IMPORTANT: Several TF2 Sourcemods required the SDK 2013 branch to be set to previous2021 to function properly, Illustrative Fortress is made on the new SDK, and requires the branch to be set to prerelease(recommended) or Default Public Version, please keep that in mind.
